# MIC-LES TTS - Portable Version

## Was ist MIC-LES TTS?
Eine moderne Text-to-Speech Anwendung mit Gaming-Integration.

## Schnellstart

### Normal verwenden:
1. Doppelklick auf `Start-MIC-LES.bat`
2. Text eingeben und "Sprechen" klicken

### Gaming-Modus:
1. Doppelklick auf `Start-Gaming.bat`
2. Hotkeys verwenden:
   - **F9**: Overlay ein-/ausschalten
   - **F10**: Text sprechen
   - **F11**: Sprechen stoppen

## Features
- 🎤 Hochwertige Text-zu-Sprache
- 🌍 Mehrsprachige Oberfläche (DE/EN/FR)
- 🎮 Gaming-Overlay mit Hotkeys
- ⚡ Portable - keine Installation nötig
- 🎨 Modernes dunkles Design

## Systemanforderungen
- Windows 10/11 (64-bit)
- 150MB+ RAM
- 200MB+ Speicherplatz
